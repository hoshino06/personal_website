# 研究テーマ

## 地域規模のエネルギーマネジメントに関わる研究

各家庭や建物で利用される電気や熱は, 電力系統や都市ガスネットワーク, 地域熱供給システムなどのインフラにより供給されています.
現在, エネルギーの発生と消費の管理は, インフラを所有・運用する事業者が一元的に行うのが一般的ですが, 近年の情報通信技術の発展に伴い, 消費者同士がインフラを利用して「エネルギーを融通しあう」という考え方が広まりつつあります.
例えば, 太陽光発電を行う家庭では, 発電電力を自家消費する（あるいは電力会社に売る）だけでなく, 近隣の家庭や建物へ売電するという可能性もあります.
このような将来を見据え、以下のような研究課題に取り組んでいます:

1. 供給システムの物理的特性を考慮したエネルギーマネジメント
　建物間でエネルギーを融通する際, その物理的な影響（例えば電力系統の電圧や周波数）は, 当事者間のみでなく, 供給システム全体に及びます.
その影響の数理モデル化, 解析, 制御によって, 供給システムの安定性を担保しつつエネルギーの融通を可能にするエネルギーマネジメントシステムの開発を目指しています. 

2. ネットワーク利用料金の設計に関する分析
　ネットワークの利用料金は、エネルギー輸送に不可欠なインフラの整備や維持のための費用を回収する役割を担うのと同時に, そのネットワークがどの程度利用されるかということにも影響します. 
ネットワークの効率的な利用を促しつつ, インフラネットワークへの適切な投資が促進されるための料金設計のあり方について検討しています.
